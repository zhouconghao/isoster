### I3: Gradient formula incorrect in linear growth mode
- **File**: `isoster/fitting.py:480`
- **Problem**: `gradient = (mean_g - mean_c) / sma / step` — for linear growth, `delta_r = step` (not `sma * step`), so dividing by `sma` is wrong.
- **Fix**: `delta_r = sma * step if not linear_growth else step; gradient = (mean_g - mean_c) / delta_r`

### I5: PA interpolation wraps incorrectly at 0/180 boundary in model.py
- **File**: `isoster/model.py:140`
- **Problem**: Linear interpolation of PA values that wrap (e.g., 170° to 10°) produces incorrect intermediate values (~90°).
- **Fix**: Apply `np.unwrap()` to PA values before interpolation.
