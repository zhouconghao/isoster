### 2. Harmonics Implementation Deep Review
- **Key discovery**: `simultaneous_harmonics=True` does NOT implement true ISOFIT (Ciambur 2015)
- Higher-order harmonics (n≥3) are computed **post-hoc only**, never entering the iterative fitting loop
- The iterative loop always uses only 5 parameters (I₀, A₁, B₁, A₂, B₂) regardless of settings
- Three consequences: geometry bias from harmonic contamination, inflated RMS, post-hoc coefficients from biased geometry
- **Full review saved**: `docs/journal/2026-02-24-harmonics-review.md`
