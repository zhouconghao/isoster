# Reference QA Figures

This directory contains reference QA figures for visual regression testing.
These figures show expected behavior for various test cases.

## Sersic Accuracy Figures

Generated by `qa/generate_sersic_figures.py`:

| Figure | Description |
|--------|-------------|
| `test_sersic_n4_noiseless.png` | n=4 de Vaucouleurs profile, noiseless, eps=0.4 |
| `test_sersic_n1_high_eps.png` | n=1 exponential profile, high ellipticity (eps=0.7), SNR=100 |
| `test_sersic_n4_eps0.6_extreme.png` | n=4 profile, eps=0.6, SNR=80, challenging case |

## Regenerating Figures

To regenerate all QA figures:

```bash
python -m qa.generate_sersic_figures
python -m qa.generate_comparison_figures
python -m qa.generate_numba_figures
```

Output goes to standardized folders under `outputs/` (gitignored). Compare with these reference figures
for visual regression testing.
