    Phase 11 Handoff: Huang2013 Campaign Fault Tolerance

    Scope

    Prepared Huang2013 workflow for all-galaxy/all-mock batch execution with failure isolation and aggregate summaries.

    Changes

    1. examples/huang2013/run_huang2013_profile_extraction.py
    2. Per-method execution is now wrapped in try/except.
    3. A failure in photutils or isoster no longer aborts the image-level extraction run.

    4. *_profiles_manifest.json now records per-method status and run_summary (requested/successful/failed methods).

    5. examples/huang2013/run_huang2013_qa_afterburner.py

    6. Missing method profile FITS no longer raises immediate FileNotFoundError.
    7. Missing outputs are recorded in method_skips.
    8. QA-render exceptions are recorded in method_failures.

    9. QA manifests now include method_summary and continue even when only one method is available.

    10. examples/huang2013/run_huang2013_campaign.py (new)

    11. Runs extraction + QA for all discovered galaxies and requested mock IDs.
    12. Continues across per-case failures and missing inputs.
    13. Writes campaign-wide summary files:
    14. JSON: huang2013_campaign_summary.json
    15. Markdown: huang2013_campaign_summary.md

    16. Tracks final counts for method outcomes (success/failed/unknown), extraction/QA invocation failures, missing inputs, and comparison QA generation.

    17. examples/huang2013/README.md

    18. Added full-campaign execution section and command for all galaxies x 4 mocks.

    Resume Command (full sample)

    uv run python examples/huang2013/run_huang2013_campaign.py \
  --huang-root /Users/mac/work/hsc/huang2013 \
  --mock-ids 1 2 3 4 \
  --method both \
  --config-tag baseline \
  --summary-dir outputs/huang2013_campaign_full
